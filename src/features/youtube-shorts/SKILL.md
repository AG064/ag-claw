# YouTube Shorts Generator

Auto-generates short vertical videos from YouTube content and publishes to social platforms.

## Setup

Requirements (installed separately):
- `yt-dlp` - YouTube downloader (`pip install yt-dlp`)
- `ffmpeg` - Video processing (`apt install ffmpeg`)

For YouTube publishing:
- `YOUTUBE_ACCESS_TOKEN` - OAuth2 access token with `youtube.upload` scope
- OR `YOUTUBE_API_KEY` - API key with YouTube Data API v3 enabled

## Usage

```typescript
import { youtubeShorts } from './features/youtube-shorts';

// Check health first
const health = await youtubeShorts.healthCheck();
if (!health.healthy) {
  console.log('Missing:', health.message);
  return;
}

// Process a YouTube video into shorts
const results = await youtubeShorts.processVideo(
  'https://www.youtube.com/watch?v=...',
  3 // number of segments
);

// Publish to all configured platforms
const publishResults = await youtubeShorts.publishShorts(
  results,
  'https://www.youtube.com/watch?v=...',
  'My Shorts'
);

for (const [key, result] of Object.entries(publishResults)) {
  if (result.success) {
    console.log(`${key}: ${result.url}`);
  } else {
    console.error(`${key}: ${result.error}`);
  }
}
```

## Configuration

```typescript
const config = {
  enabled: true,
  outputDir: './outputs/shorts',
  maxDuration: 60,        // max seconds per short
  quality: '1080p',      // or '720p'
  platforms: ['telegram', 'twitter', 'youtube'],
  voiceover: false,
  youtube: {
    privacyStatus: 'unlisted',  // 'public', 'unlisted', or 'private'
    titleTemplate: '{baseTitle} - {caption}',
    descriptionTemplate: 'Generated short from {url}\n\nGenerated by AG-Claw',
  },
};
```

## YouTube Upload

YouTube upload uses the YouTube Data API v3 with OAuth2 or API key authentication.

### Setting up YouTube API:

1. Go to Google Cloud Console → APIs & Services → YouTube Data API v3
2. Enable the API
3. Create credentials (API Key or OAuth2 Client ID)
4. Set environment variable:
   ```bash
   export YOUTUBE_ACCESS_TOKEN="ya29....."  # OAuth2 token
   # OR
   export YOUTUBE_API_KEY="AIza....."      # API Key
   ```

### OAuth2 Setup (for uploading to your channel):

1. Create OAuth2 Client ID (Desktop app or Other)
2. Get authorization code:
   ```
   https://accounts.google.com/o/oauth2/v2/auth?client_id=YOUR_CLIENT_ID&response_type=code&scope=https://www.googleapis.com/auth/youtube.upload&redirect_uri=urn:ietf:wg:oauth:2.0:oob
   ```
3. Exchange code for tokens
4. Set `YOUTUBE_ACCESS_TOKEN` and `YOUTUBE_REFRESH_TOKEN`

## Workflow

1. Download full YouTube video
2. Analyze frames to find best moments (scene changes)
3. Cut 9:16 vertical clips
4. Add captions
5. Publish to configured platforms (YouTube Shorts, Telegram, Twitter)

## Limitations

- Requires yt-dlp and ffmpeg installed
- YouTube ToS prohibits downloading some content
- Feature is disabled by default
- Scene detection is simplified (evenly distributed segments)
- YouTube upload requires API credentials
